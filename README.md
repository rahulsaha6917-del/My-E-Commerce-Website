## 📌 About The Project

This project is an **E-Commerce Website** where users can:

✅ Browse products categorized by type  
✅ View product details  
✅ Add/remove products from the cart  
✅ Enjoy a clean UI & responsive experience  
✅ Experience dynamic cart-management

-------
📌 Perfect for practising React fundamentals such as:


✔ State Management  
✔ Props & Components  
✔ Routing  
✔ API integration (optional)
